Patern fo decorartor
def my_decorator(fn):
    print("Этот код будет выведен один раз в момент декорирования функции")
    def wrapper(*args, **kwargs):
        print('Этот код будет выполняться перед каждым вызовом функции')
        result = fn(*args, **kwargs)
        print('Этот код будет выполняться после каждого вызова функции')
        return result
    return wrapper

git status - вывести статус текущего проекта
git log - показат ьисторию изменений
git add name - добавить файл в область подготовки данных/проекта
git commit keys - создать коммит (версию) текущего набора/проекта
	keys:
	-m сохранение коммита с описанием
	-a При выполнении команды с флагом -a будет сделан снимок всех изменений в рабочем каталоге. Это включает в себя только изменения отслеживаемых файлов (то есть тех, которые были добавлены с помощью git add ранее).
	-am Сочетание флагов -a и -m создаёт коммит всех изменений с сообщением для коммита.
	--amend Выбранные изменения будут добавлены в последний коммит. Команда также вызовет открытие системного текстового редактора для внесения изменений в сообщение коммита. Эту команду часто используют, если была допущена ошибка.
git status - проверить статус проекта
git branch keys name - создать ветку name для проекта
	keys
	-m old_name new_name - переименование ветки из old_name в new_name
	-d branch_name	- удаление ветки с имененм branch_name
	--list - вывести список веток
	-v - вывести последнии коммиты ветки
