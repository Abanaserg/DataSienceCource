Модуль "Разведывательный анализ данных"

Основные задачи проекта:

- 	Провести анализ данных, оценив их качество и адекватность
- 	При необходимости очистить данные 
- 	Сформировать гипотезы для дальнейшего построения модели формирования предсказуемой величины (score)
- 	Определиться с параметрами итоговой модели

Краткая информация о данных:

- 	После очистки данных (выбросы, статистически не зависимые данные, ошибки заполнения) наблюдается 4.5 % 
	пустых значений. Однако, как итог только 96 строк не имеют пустых значений вообще.
- 	Выбросы и ошибки при заполнении наблюдались в данных по образовании отца (father_education), семейных отношениях (family_relations), прогулах (absence).
- 	Данные по оценкам (score) имели пустые значения и нули. Принято решение об исключении этих строк.
- 	В данных по возрасту (age) и образованию родителей (mother_education, father_education) имелись значения, которые редко встречались
	в общем массиве (менее 5 значений). Были заменены на пустые.
-	Данные по образованию родителей имеют довольно высокую корреляцию (0.63) образование родителей зачастую совпадают.
-	Данные по образованию матери имеют более высокую корреляцию с её профессией, нежели аналогичные данные по второму родителю
-	Некоторые данные коррелируют между собой при дополнтельном разбиении на меньшие фрагменты данных. например в школе GP гораздо больше ученико из города, 
	чем из пригорода, когда в школе MS количество учеников по данному параметру примерно одинаковое.
- 	Учитывая задачу по определению влияния параметров массива данных на общую оценку определены основные и второстепенные срезы данных:
		а) 	основную статистически значимую информацию несут данные по образованию родителей (mother_education, father_education), 
			проффесия матери (mother_job), время на учебу вне школы (study_time), количество внеучебных неудач (failures), 
			дополнительная образовательная поддержка (schools_up), и свободное время (go_out), пропуски занятий (absence)
		б) 	меньшую статистически значимую информацию несут данные по дополнительным внеучебным занятиям (activities)
			возрасту (age), семейным отношениям (family_relations), размеру семьи (family_size), семейной поддержки (familys_up), работе отца (father_job), 
			количеству свободного времени (free_time), опекуну (guardian), состоянию здоровья (health), посещению детского сада (nursery), 
			наличию платных курсов(paid_courses), статуса родителей (parent_status), причины выбора школы (reason), наличию романтических отношений (romantic), 
			школы (school), пола (sex), планирование своего будущего (thought_higher_eduaction), времени в пути до школы (travel_time) 
			(смотри информацию переменной meaning_datas_d2)
-	Положительное влияние на успеваемость оказывают следующие параметры: образование родителей (mother_education, father_education) и время на учебу вне школы (study_time)
-	Отрицательное влияние на успеваемость оказывают следующие параметры: возраст (чем старше, тем ленивее:)) (age), 
	количество внеучебных неудач (failures), свободное время (go_out)

Этапы работы над проектом:

- Первичный анализ данных. Проверка на адекватность и выбросы
- Очистка данных по результатам первичного анализа
- Проверка значимости данных каждого параметра на общую успеваемость (построение гипотез о влиянии па успеваемость)
- Проверка значимости отдельных срезов данных на общую успеваемость
- Очистка данных по результатам второй итерации по анализу значимости параметров
- Формирование выводов для дальнейшего построения модели
